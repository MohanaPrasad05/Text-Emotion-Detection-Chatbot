# Emotion Analysis Chatbot

An AI-powered chatbot that detects emotions from text and presents:
- Primary and secondary emotions
- Emotion intensity percentage
- Confidence level
- Emotion-aware chatbot responses
- (Optional) Emotion drift across long text

⚠️ Disclaimer: This system provides emotional interpretation, not medical or psychological diagnosis.

## Project Structure
- training/ : ML training notebook
- model/    : Saved ML model and vectorizer
- backend/  : API and logic
- frontend/ : Chatbot UI

## How to Run
1. Install backend dependencies
2. Run backend server
3. Open frontend in browser
